# Grades

For a given grade percentage print the grade letter according to the following rules:

* **A**: 90-100%
* **B**: 80-89%
* **C**: 70-79%
* **D**: 60-69%
* **F**: 0-59%

**Example:**

For the given `int n = 82` the out put should be `B`
